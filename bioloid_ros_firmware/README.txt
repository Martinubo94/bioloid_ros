To use the project template:

1)  Open Eclipse

2)  Select File->New->Makefile Project with Existing Code

3)  Click the "Browse" button to the right of the Existing Code Location field and select the folder that contains the project template, then name the Project and click finish.

4)  Build by selecting Project->Clean... and then Project->"Build Project"

5)  Use RoboPlus Terminal application to load the CM530.bin file in to CM-530 firmware according to the instructions on the Robotis website for Embedded-C programming
